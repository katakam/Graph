Basic Help

If you want to search for a Service Request,

  • Go to the service request page.
  • Enter the case ID if you know it.
  • Click on Search.
  • If you don't have the ID, type in some keywords associated with the SR and search
  • If you want to do Multi value search click on the check box for multi value and enter the fields other than ID and keywords.

If you want to search for a Problem Report,

  • Go to the problem report page.
  • Enter the PR ID if you know it.
  • Click on Search.
  • If you don't have the ID, type in some keywords associated with the PR and search
  • If you want to do Multi value search click on the check box for multi value and enter the fields other than ID and keywords.
